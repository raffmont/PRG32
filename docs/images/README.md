# Screenshot Placeholders

Add screenshots with these names:

- `prg32-qemu-game.png` - QEMU window running a demo game
- `prg32-upload-success.png` - terminal output from `prg32_game.py upload-qemu` or `upload`
- `prg32-runtime-api.png` - `curl http://192.168.4.1/api/runtime` response

Capture tips:

- Keep terminal width around 100 columns.
- Include command + output in the same image when possible.
- Use PNG format and avoid heavy compression artifacts.
